Introduction to CSS

CSS (Cascading Style Sheets) is used to style and layout web pages. For example, it can change the font, color, and spacing of your content.

Basic Syntax

selector {
    property: value;
}

Example

body {
    font-family: Arial, sans-serif;
}

h1 {
    color: blue;
}

p {
    font-size: 16px;
}

Common Properties

  • color: Sets the color of text
  • background-color: Sets the background color of an element
  • font-size: Sets the size of the text
  • margin: Sets the margin outside an element
  • padding: Sets the padding inside an element
